# Proyectos de Estudio

Repositorio que recopila diferentes proyectos que desarrollé mientras aprendía sobre **procesamiento, análisis de datos, machine learning y visión por computador**.

El objetivo de este repositorio es dejar registro de ejercicios, pruebas, notebooks y experimentos realizados durante mi proceso de aprendizaje, abordando temas como clasificación, regresión, clustering, detección de objetos y procesamiento de imágenes.

## 📌 Descripción

Este repositorio funciona como una colección de proyectos académicos y personales orientados al estudio práctico de técnicas de análisis de datos.  
Cada carpeta contiene implementaciones, notebooks, datasets o documentos asociados a distintos tópicos explorados durante el aprendizaje.

No está pensado como una librería final o un producto terminado, sino como un espacio de práctica, experimentación y documentación del proceso de estudio.

## 📁 Contenido del repositorio

| Carpeta | Descripción |
|---|---|
| `Deeplearning/` | Experimentos relacionados con redes neuronales, regresión, datasets y análisis mediante notebooks. |
| `Sensado con Yolo/` | Proyecto relacionado con detección de objetos usando YOLO, incluyendo notebook de entrenamiento y documentación asociada. |
| `kmeans/` | Implementaciones y pruebas del algoritmo K-Means/K-Means++, aplicadas a análisis, clustering y procesamiento de imágenes. |
| `figures/` | Figuras e imágenes utilizadas como material de apoyo para los proyectos. |

## 🧠 Temas abordados

- Procesamiento y análisis de datos.
- Machine Learning.
- Deep Learning.
- Clasificación y regresión.
- Clustering con K-Means.
- Inicialización K-Means++.
- Procesamiento de imágenes.
- Detección de objetos con YOLO.
- Uso de notebooks para experimentación y visualización.

## 🛠️ Tecnologías utilizadas

Algunos de los proyectos pueden utilizar herramientas y librerías como:

- Python
- Jupyter Notebook
- NumPy
- PyTorch
- scikit-image
- MATLAB
- YOLO

> Las dependencias exactas pueden variar según cada proyecto o notebook.
